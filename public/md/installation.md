# Installing the components

> For the complete documentation index, see [llms.txt](/llms.txt)

## INSTALLATION

The components use a shadcn/ui registry to distribute them, so you will need to install the shadcn package first. Check the official [documentation page](https://ui.shadcn.com/docs/installation) for your preferred framework.

Once shadcn is configured for your environment, you can install each component through the shadcn command line like so:

```
bunx --bun shadcn@latest add https://ui.bartoszbak.org/r/cornered-button.json
```

This will install the cornered button component, which you can check out in more detail here: [cornered button](/components/cornered-button.md).
